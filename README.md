# 🚀 AI Recipe Generator App
A full-stack AI-powered cooking assistant that helps users generate recipes based on available pantry items, plan meals, and manage groceries efficiently.

Built using Node.js, React, PostgreSQL, and Generative AI (Google gen ai), this application provides a smart and personalized cooking experience.


## 🔧 Tech Stack
Frontend: React + Tailwind CSS v4
Backend: Node.js (Express)
Database: PostgreSQL (Neon)
Authentication: JWT + bcryptjs
AI Integration: Google gen ai

## ✨ Features
🔐 Authentication
- Secure user signup & login using JWT
- Password hashing with bcryptjs
## 🥫 Pantry Management
- Track ingredients with expiry dates
- Get low-stock alerts & expiry notifications
## 🤖 AI Recipe Generation
- Generate recipes from available pantry items
- Powered by Google Gemini AI
## 🥗 Filters & Preferences
- Dietary filters: Vegetarian, Vegan, Keto, Paleo, Gluten-Free
- Cuisine selection (10+ cuisines)
- Save default preferences for quick generation
## 📖 Recipe Management
- View ingredients, step-by-step instructions
- Nutrition information & AI cooking tips
- Save and organize recipes
## 📅 Meal Planner
- Weekly planner (Breakfast, Lunch, Dinner)
- Easy week navigation
## 🛒 Shopping List
- Add items from recipes
- Mark items as purchased
- One-click add to pantry
## 🧠 Smart Personalization
- User preferences auto-fill recipe generator
## 🗄️ Database Design
- Well-structured normalized PostgreSQL schema
## 🎨 UI/UX
- Fully responsive modern UI using Tailwind CSS
